description: Push changes to GitHub to trigger Vercel deployment
---

1. Stage all changes
   ```bash
   git add .
   ```

2. Commit changes (User should provide a message, or I will generate one)
   ```bash
   git commit -m "Update: [Brief description of changes]"
   ```

3. Push to origin
   ```bash
   git push origin main
   ```

4. 작업할 때마다 commit 메시지 남기고, github에 푸시해.
